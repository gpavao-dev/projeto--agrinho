## ✨ Sobre o Projeto

O **Agrinho** é um programa de educação do campo que há mais de 25 anos transforma a vida de milhares de crianças e jovens rurais brasileiros. Este site foi desenvolvido para:

- Apresentar o programa de forma moderna e atrativa
- Facilitar o acesso a materiais educativos
- Divulgar as ações, concursos e resultados
- Conectar escolas, professores, pais e alunos

## 🚀 Funcionalidades

- ✅ Página inicial vibrante com animações
- ✅ Seção "O que é Agrinho?"
- ✅ Galeria de desenhos e histórias premiadas
- ✅ Materiais para download (livros, cartilhas, atividades)
- ✅ Mapa interativo das escolas participantes
- ✅ Histórias de sucesso e depoimentos
- ✅ Área do Professor e do Aluno
- ✅ Responsivo e acessível
- ✅ Tema sustentável com cores da natureza

## 🛠 Tecnologias Utilizadas

- **Frontend:** Next.js 15 (App Router) + React + TypeScript
- **Estilização:** Tailwind CSS
- **UI:** Shadcn/ui + Radix UI + Framer Motion
- **Ícones:** Lucide React
- **Imagens:** Otimizadas com Next/Image
- **Deploy:** Vercel

## 📸 Visual

O site utiliza tons de **verde, amarelo, marrom e azul**, transmitindo a energia do campo, da natureza e da infância.
